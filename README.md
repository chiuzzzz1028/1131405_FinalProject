# 1131405_FinalProject_學生作業與考試管理系統

## 專案介紹

這個學生作業與考試管理系統主要用來協助學生整理不同科目的作業、考試、報告與專題。使用者可以記錄課業名稱、截止日期、優先程度及完成狀態，並透過搜尋、篩選、與排序等功能，快速掌握目前的課業進度。

## 功能介紹

### 1. 新增課業項目

使用者可在左側資料輸入區填寫以下資料：

- 科目
- 類型
- 名稱
- 截止日期
- 優先程度
- 狀態

按下「新增」後，資料會加入中間的課業清單。

### 2. 修改項目

點選課業清單中的資料後，內容會自動顯示於左側輸入區。修改資料後按下「修改」，即可更新原本的項目。

### 3. 刪除項目

點選要刪除的資料後按下「刪除」，系統會先顯示確認訊息，避免使用者誤刪資料。

### 4. 清空輸入欄位

按下「清空」可清除左側目前輸入或選取的資料，並恢復各欄位的預設值。

### 5. 搜尋與篩選

使用者可以輸入關鍵字搜尋科目或項目名稱，也可以依照以下條件篩選資料：

- 狀態：全部、未開始、進行中、已完成
- 類型：全部、作業、考試、報告、專題、其他

按下「顯示全部」後，可取消搜尋與篩選條件並重新顯示所有資料。

### 6. 排序功能

系統提供以下排序方式：

- 依截止日期排序
- 依優先程度排序

優先程度的排序順序為：

`高 → 中 → 低`

若優先程度相同，則再依截止日期排列。

### 7. 雙擊表示完成

在課業清單中雙擊尚未完成的項目，確認後即可將該項目的狀態快速改為「已完成」。

若該項目原本已經完成，系統會顯示提醒訊息。

### 8. 統計與到期提醒

視窗下方的狀態列會顯示：

- 總項目數
- 未完成數量
- 已完成數量
- 已逾期數量
- 即將到期數量

逾期項目的判斷方式為：

> 截止日期早於今天，且狀態不是「已完成」。

即將到期項目的判斷方式為：

> 截止日期介於今天至未來三天內，且狀態不是「已完成」。

### 9. 顏色提示

課業清單會依照項目狀態與到期情況顯示不同顏色：

| 顏色 | 代表狀態 |
|---|---|
| 綠色 | 已完成 |
| 紅色 | 已逾期 |
| 黃色 | 三天內即將到期 |
| 藍色 | 進行中且尚未接近截止日期 |
| 白色 | 未開始且尚未接近截止日期 |

### 10. 儲存與讀取檔案

系統支援將目前資料儲存成 CSV 檔案，也可以讀取先前儲存的 CSV 檔案。

CSV 檔案包含以下欄位：

```text
科目,類型,名稱,截止日期,優先程度,狀態
```

讀取檔案時，系統會檢查：

- 檔案是否為空
- 每一行的欄位數量是否正確
- 截止日期格式是否正確

若檔案內容不符合格式，系統會顯示錯誤訊息，而不會直接關閉程式。

## 快捷鍵

| 快捷鍵 | 功能 |
|---|---|
| `Alt + H` | 開啟說明選單 |
| `Alt + F` | 開啟檔案選單 |
| `F1` | 顯示操作說明 |
| `Ctrl + O` or `Alt + F` 後按 `O` | 讀取檔案 |
| `Ctrl + S` or `Alt + F` 後按 `S` | 儲存檔案 |
